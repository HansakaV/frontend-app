## Description
The frontend provides a dashboard to manage labours, buses, and bookings. It communicates with the backend services through the API Gateway.

## Technology Stack
- React / Vite
- Tailwind CSS (Optional) / Vanilla CSS
- Axios for API calls

## Setup Instructions
1. Navigate to the directory: `cd frontend-app`
2. Install dependencies: `npm install`
3. Run in dev mode: `npm run dev`
